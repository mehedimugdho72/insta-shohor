2. Local storage VS Session storage

Answer: Local Storage - stores data without an expiration date and clears only via JavaScript, or clears browser cache / locally stored data. The transmission limit is the highest of the three.
Session Storage - Session storage The session storage property allows you to access a session storage object. Data stored in LocalStore has no expiration time setting, and data stored in session Store is cleared at the end of the page session except the page session is maintained while the browser is open and reloading or restoring the page still retains the original page session. Opening a page in a new tab or window will start a new session, which is different from how session cookies work.
